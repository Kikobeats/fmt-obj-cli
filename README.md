## Install

```bash
$ npm install fmt-obj-cli --global
```

## Usage

```bash
$ fmt --help

Usage
  $ fmt-obj <filepath>

Examples
  $ fmt-obj package.json
  $ curl https://api.github.com/repos/expressjs/express | fmt-obj
```
